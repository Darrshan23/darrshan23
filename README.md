### 🔐 Go-HIDS — Real-Time Host Intrusion Detection System
> **Jan 2026** · Go · Goroutines · Regex · MITRE ATT&CK · Webhook

A production-grade security agent built in Go that monitors host logs in real time using concurrent Goroutines (non-blocking). Detects IOCs such as SSH Brute Force (MITRE T1110) and privilege escalation via a Regex engine, then pushes structured JSON alerts via webhooks to analyst dashboards.

---

### 🌐 Web Exploitation & Pen Testing Lab
> **Jan 2026 – Present** · Kali Linux · WebSploit · MITM · Credential Harvesting

A self-hosted virtualized environment replicating real-world web application vulnerabilities. Attack simulations include Man-in-the-Middle attacks, directory scanning, and credential harvesting. All traffic is captured and analysed to build detection rules and improve threat hunting playbooks.

---

### 🖥️ Personal Mini SIEM Ecosystem
> **Self-Directed Lab** · Wazuh · Log Correlation · Detection Engineering

End-to-end SIEM deployment ingesting, parsing, and correlating operational telemetry from multiple log sources. Used to practice fine-tuning alert thresholds, writing detection rules, and building SOC-style dashboards.

---

### 🤖 IoT Obstacle Avoidance Car
> **April 2025** · Python · Arduino · Raspberry Pi · IoT

Autonomous embedded system with real-time obstacle detection. Microcontrollers programmed using Arduino, integrated with Raspberry Pi for sensor processing and motion control via Python.
